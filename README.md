# 🚀 Portfolio Website

My digital space to showcase my skills, projects, and what I love working on.

## 📋 About

This is a modern, responsive portfolio website built with cutting-edge web technologies. It features my professional background, project showcases, and ways to get in touch.

---

## ✨ Features

- **Responsive Design** - Works seamlessly on all devices
- **Modern UI** - Clean and professional interface
- **Project Showcase** - Display your best work
- **Contact Section** - Easy ways for visitors to reach you
- **Performance Optimized** - Fast load times and smooth interactions
- **SEO Friendly** - Optimized for search engines

---

## 🛠️ Tech Stack

This project is built with:

- **Vite** - Lightning-fast frontend build tool
- **React** - UI library for building interactive components
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS** - Utility-first CSS framework
- **shadcn-ui** - High-quality UI components

## 🚀 Getting Started

### Prerequisites

- Node.js & npm installed ([install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating))

### Installation & Setup

```bash
# Clone the repository
git clone https://github.com/aritramukhopadhyay2004/new-portfolio-website.git

# Navigate to project directory
cd new-portfolio-website

# Install dependencies
npm install

# Start development server
npm run dev
